Concept / Portfolio Project

Teton Food Truck Website Site Plan

Project Overview

Teton Food Truck is a single food truck business. This website is a concept project created to demonstrate how a food truck can present its menu, ordering flow, and location online.

Purpose

Allow customers to view the food truck menu, place a demo order, and find the truck’s location using an interactive Google Map.

Target Audience

  • Local residents
  • Tourists
  • Event attendees

Site Structure

  • Hero
  • About
  • Menu
  • Order
  • Location

Design Direction

  • Simple and mobile first
  • Clear menu presentation
  • Minimal distractions

Disclaimer

All ordering features are simulated. No real orders or payments are processed.
